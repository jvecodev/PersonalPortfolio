EasyStock

🧑‍🔬 O EasyStock é um projeto que consite em ums sistema de copntrole de estoque desenvolvido e python, utilizando a biblioteca Tkinter para interface gráfica e MySQl para gerenciamento do banco de dados. 🏅 O sistema permite o cadastro de novas categorias de produtos.Os produtos podem ser adicionados ao estoque, informando detalhes como nome, descrição, quantidade, preço de compra e venda, e a categoria a qual pertencem.A interface gráfica foi projetada para ser intuitiva e prática, com campos de entrada e botões de ação. Conta também com um sistema de rolagem para acomodar a visualização de grandes quantidades de dados.

🧑‍🔬 EasyStock is a project that consists of a stock control system developed in Python, using the Tkinter library for the graphical interface and MySQL for database management. 🏅 The system allows the registration of new product categories. Products can be added to the stock by providing details such as name, description, quantity, purchase price, sale price, and the category they belong to. The graphical interface was designed to be intuitive and practical, with input fields and action buttons. It also features a scrolling system to accommodate the display of large amounts of data.
